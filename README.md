📖 NarrativeFlow: Interactive Story Co-Writer

NarrativeFlow is an AI-powered storytelling application that helps users create, edit, and enhance stories interactively. It combines a clean writing interface with a smart AI co-writer powered by Ollama  to assist in generating ideas, dialogues, and story continuations.

Features

✍️ Interactive Story Writing     

Write and edit your manuscript in real time
Clean and distraction-free writing interface

🤖 AI Co-Writer Assistant
Ask for story ideas, plots, or continuations
Powered by local AI using Ollama (TinyLlama)

💬 Chat-Based Assistance
Maintain chat history for each story
Append AI responses directly to your manuscript

👤 User Authentication
Register and login system
Guest mode available

💾 Persistent Storage
Stories and users saved using data.json
Data remains even after restarting the app

📚 Multiple Story Management
Create and switch between stories
Search stories by title

📥 Export Feature
Download stories as .txt files

🛠️ Tech Stack
Frontend & Backend: Streamlit

AI Model: Ollama 

Language: Python

Storage: JSON file (data.json)
